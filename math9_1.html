1. Fundamental Concepts in Set Theory

1.1 Definition of a Set

A set is a well-defined collection of distinct objects, considered as an object in its own right. These objects are called elements or members of the set.

Sets are typically denoted by capital letters (A, B, C,...) and elements by lowercase letters (a, b, c,...).

1.2 Representation of Sets

Sets can be represented in several ways:

  1. Roster/Tabular Form: Listing all elements between curly braces {}.
    Example: A = {1, 2, 3, 4, 5}
  2. Set-Builder Notation: Defining a set by a property its members satisfy.
    Example: B = {x | x is an even integer and x > 0}
  3. Descriptive Form: Describing the set in words.
    Example: C = the set of all prime numbers less than 20

2. Types of Sets

2.1 Basic Set Classifications

  • Finite Set: A set with a countable number of elements. Example: A = {1, 2, 3}
  • Infinite Set: A set with an uncountable number of elements. Example: N = {1, 2, 3, ...}
  • Empty/Null Set (‚àÖ): The unique set containing no elements.
  • Singleton Set: A set with exactly one element. Example: {5}
  • Universal Set (U): The set containing all objects under consideration.

2.2 Set Relationships

  • Equal Sets: Two sets A and B are equal (A = B) if they contain exactly the same elements.
  • Equivalent Sets: Two sets are equivalent if there exists a bijection between them (they have the same cardinality).
  • Subset (A ‚äÜ B): Every element of A is also in B.
  • Proper Subset (A ‚äÇ B): A is a subset of B but A ‚â† B.
  • Power Set P(A): The set of all subsets of A.

7. Advanced Topics in Set Theory

7.1 Axiomatic Set Theory (ZFC)

The standard foundation of mathematics includes these axioms:

  1. Extensionallity
  2. Regularity
  3. Specification (Subset)
  4. Pairing
  5. Union
  6. Replacement
  7. Infinity
  8. Power Set
  9. Choice (AC)

7.2 Paradoxes and Independence Results

  • Russell's Paradox: Shows that the "set of all sets" leads to a contradiction
  • Banach-Tarski Paradox: A ball can be decomposed and reassembled into two identical balls (using AC)
  • Independence of CH: The continuum hypothesis cannot be proved or disproved from ZFC

7.3 Alternative Set Theories

  • Von Neumann-Bernays-G√∂del (NBG): Allows proper classes
  • New Foundations (NF): Attempts to avoid paradoxes differently
